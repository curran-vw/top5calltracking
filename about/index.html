About me

Hi, I'm Lisa.

I run a small lead-gen agency in Atlanta and operate a handful of rank-and-rent properties on the side. This site is where I write about the call tracking tools I actually use.

The short version

I've been doing lead-gen marketing since 2017. Started as an in-house marketer at a home-services franchise, left in 2019 to start my own thing, and have been running my own agency and rank-and-rent properties ever since.

Since 2019 I've deployed call tracking on more than 50 different setups: client landing pages, GMBs, pay-per-call campaigns, rank-and-rent properties. I've onboarded, ported numbers between, and ripped out at least a dozen different tools across that time. The product I had on day one isn't the one I use today.

Why I started this site

Almost every "best call tracking software" roundup I've read has been written by someone who's never used the tools they're recommending. They're written from spec sheets and vendor decks. The result is a list that's technically accurate and practically useless.

I write from the seat I'm in. My #1 pick is the tool I run on every property I own. The other four are tools I'd recommend in specific situations where my top pick isn't the right answer for that exact buyer. I've written down where each one wins and where each one falls short, with the actual numbers from my P&L.

How I test

I run real campaigns through every tool I review. I time the setup. I provision real numbers, route real calls, and check the attribution data against what I know to be true from my own ad accounts. I score on four things that matter when you're actually doing the work: cost, setup time, daily usability, and support.

Full methodology is on the how I tested page. The short version: nothing on this site is theoretical, and I won't write up a tool I haven't actually used in production.

How this site makes money

I earn an affiliate commission when you sign up for a tool through a link on this site. That's how almost every independent review property in this space is funded. Two things to know:

Commissions don't change my rankings. My #1 pick is the tool I'd run regardless of affiliate economics, because I run it on my own properties. If a vendor offered me a richer affiliate deal in exchange for a higher rank, I'd say no, because the credibility of the site is worth more than any single payout.

I disclose this on every page. The footer of every page says I earn affiliate commissions, and the FTC requires it. I'm an independent operator. I'm not owned by, employed by, or affiliated with any of the products I review beyond the standard affiliate-link relationship.

Where I'm wrong

If you spot a stale price, a feature change I missed, or a fact I got wrong, please email me. I'd rather correct the page than leave it standing. Same email if you've used a tool I should be reviewing and I'm not. I read everything that comes in.

The contact page has my email.

What's next on this site

I refresh the rankings whenever pricing or features change in a way that would shift the order. Major re-ranks happen around twice a year. The "Updated" date on each page reflects the most recent edit.
